# Electronic Supplementary Material (ESM)

Este diretório consolida o material suplementar exigido para submissão.

O repositório público com os artefatos de reprodutibilidade (strings de busca, corpus bibliográfico em BibTeX, scripts de triagem e rotinas analíticas, além de resultados intermediários) está disponível no OSF sob DOI https://doi.org/10.17605/OSF.IO/J7STC.

Para fins de submissão, este ESM deve incluir, no mínimo, os scripts e datasets utilizados para gerar as figuras, tabelas e métricas reportadas no manuscrito, com instruções de reprodução e identificação clara das entradas e saídas.
