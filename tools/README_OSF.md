Este diretorio contem utilitarios para enviar artefatos do workspace ao OSF usando o osfclient.

O fluxo mais seguro e manter o repositorio Git como trilha de mudanca e publicar no OSF apenas um pacote de artefatos reproduziveis, incluindo manuscrito, strings de busca, exports BibTeX, scripts e relatorios.

Os comandos usam autenticacao por token via variavel de ambiente OSF_TOKEN e o identificador do projeto OSF via variavel OSF_PROJECT.

O identificador do projeto e o sufixo do DOI, por exemplo no DOI https://doi.org/10.17605/OSF.IO/J7STC o ProjectId a ser usado e J7STC.
